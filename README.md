# HR Analytics Dashboard | Power BI

## Project Overview
This project analyzes HR data to help organizations understand
employee attrition, workforce distribution, and key HR trends
using Power BI.

## Key Features
- Employee Attrition Analysis
- Department-wise Employee Count
- Age & Experience Distribution
- Gender Diversity Analysis
- Interactive Filters

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
